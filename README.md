# 🌿 Daily Reflection Tree

A deterministic reflection system that models human thinking patterns using a structured decision-tree approach instead of generative AI.

---

## 🧠 Project Overview

This project implements a guided reflection tool where users answer a series of structured questions. Each response leads to a predefined path, ensuring consistent and explainable outcomes.

The system evaluates user behavior across three key psychological dimensions:

- **Agency** → Internal vs External control  
- **Contribution** → Contribution vs Entitlement  
- **Perspective** → Self vs Others  

---

## 🚀 Features

- ✅ Deterministic decision-tree logic (no randomness)
- ✅ 25+ structured nodes
- ✅ Multi-step interactive UI
- ✅ Insight-based summary generation
- ✅ Restart functionality
- ✅ Clean and responsive design

---

## ⚙️ Tech Stack

- **Frontend:** HTML, CSS, JavaScript  
- **Logic Structure:** JSON (decision tree)  
- **Backend (optional):** Python (CLI version)

---

## 🧩 How It Works

1. User starts reflection session  
2. Answers guided questions  
3. Each answer maps to a predefined node  
4. Signals are collected across 3 axes  
5. Final summary + insight is generated  

---

## 🎯 Key Idea

> The intelligence of the system is encoded in structured decision paths rather than generated dynamically.

This ensures:
- Consistency  
- Explainability  
- Reliability
